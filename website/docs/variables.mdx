## \*\*\* Variables \*\*\* Section

Variables which are defined in the \*\*\* Variables \*\*\* section are available in all test cases and keywords in the same file.  
Variables defined in the \*\*\* Variables \*\*\* section are **global** variables.  
If a `.resource` or a `.robot` file with a \*\*\* Variables \*\*\* section is imported, the variables are also available globally.

Example:

```robotframework
*** Variables ***
${my_var}     my_value
@{my_list}    Apple    Banana    Orange
&{my_dict}    name=my_value1    password=my_value2

*** Test Cases ***
Test Case 1
    Log    ${my_var}
    Log    ${my_list}
    Log    ${my_dict}
    FOR    ${item}    IN    @{my_list}
        Log    ${item}
    END
    Log    ${my_dict}[name]
    Log    ${my_dict}[password]
    FOR   ${key}    ${value}    IN    &{my_dict}
        Log Many    ${key}    ${value}
    END
```
